REST API Documentation

This project is a REST API built using Express.js and integrated with MongoDB via Mongoose. The API provides endpoints for managing users and courses, utilizing JWT authentication to secure access to protected resources.

Base URL

All API endpoints should be prefixed with:

https://satisfactory-teddy-abdalrhmangroup-f18e5404.koyeb.app/api/

Technologies Used

  • Node.js
  • Express.js
  • MongoDB & Mongoose
  • JWT (jsonwebtoken)
  • bcrypt.js
  • dotenv
  • morgan
  • CORS
  • Multer

Project Setup

1. Install Dependencies

npm install express mongoose dotenv jsonwebtoken bcryptjs cors morgan multer

2. Environment Variables

MONGO_URL=your_mongodb_connection_string
JSONWEBTOKEN_SECRET_KEY=your_secret_key
PORT=5000

3. Start the Server

node index.js

API Endpoints

Course Management

1. Get All Courses

GET /api/courses

2. Get a Single Course

GET /api/courses/:courseId

3. Create a New Course

POST /api/courses

4. Update a Course

PATCH /api/courses/:courseId

5. Delete a Course

DELETE /api/courses/:courseId

User Management

1. Get All Users

GET /api/users

2. User Registration

POST /api/users/register

3. User Login

POST /api/users/login

File Uploads

Multer is used for handling file uploads.

Authentication & Authorization

Users receive a JWT token upon registration or login.

Error Handling

  • Validation Errors: 400 Bad Request
  • Unauthorized Access: 401 Unauthorized
  • Resource Not Found: 404 Not Found

Conclusion

This REST API provides a secure and scalable way to manage courses and users.
